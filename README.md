# Smashrun custom compontent for Home Assistant
Smashrun integration for Home Assistant

## Install

Copy `smashrun` folder to `custom_components` folder in your config folder.

Restart HA.

## Setup


1. Configuration > Integration > Add Integration > Smashrun
2. Enter your Smashrun API token
